HTML

  • HTML is one of the languages used to build websites
  • HTML uses elements which provide content and frame websites
  • Heading elements are ranked in order of importance
  • The documents title is specified between the title tags within the head section
  • HTML metadata is defined in the head section of HTML

CSS

  • CSS Cascading Style Sheet is used to control the presentation of elements within HTML
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Each style rule comprises a selector and declaration block
  • Each declaration specifies a property and a value to be applied

Git

  • git status: checks which branch we are currently on
  • git pull origin main command ensures that the one has the latest version of the project
  • git add -A adds modifications in the current working branch
  • git -h is a command is a help command. It will display what a flag does
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • Functions are the building blocks of JavaScript
  • JavaScript is linked to the body element
  • JavaScript helps users to interact with a website
  • The order of the code determines the outcome!
  • We can use conditional statements and loops to control the order in which code is executed, or if it is even executed at all.
